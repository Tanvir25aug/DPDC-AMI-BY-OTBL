🟢 CI_ — Customer Care & Billing

Core billing system tables. Manages the entire customer lifecycle: persons, accounts, service agreements, premises, service points, bills, payments, adjustments, and financial transactions.

1,109 total 692 core ~150 CHAR ~100 _L lang

🟣 D1_ — Meter Data Management

MDM system tables. Manages physical meter devices, measurement components (registers), meter readings, usage calculations, network facilities (feeders/substations), and AMI head-end integration.

439 total 178 core

🟡 F1_ — Oracle Utilities Framework

Cross-cutting framework tables. Service tasks (work items), outbound messaging, data migrations, synchronization requests, batch processing, audit logs, and system configuration.

265 total 120 core

🔵 C1_ — CC&B Extended (C2M)

Extended CC&B features: customer service requests, marketing campaigns, lead management, notification preferences, rate calculations, prepaid metering, and customer portal users.

264 total 114 core

🔴 W1_ — Work & Asset Management

Field operations and asset lifecycle management. Manages physical assets (transformers, poles, cables), work orders, field activities, crews, timesheets, purchase orders, inventory, and compliance tracking.

835 total 349 core
